# Ayurveda Ayatanam Website

Welcome to the website project for **Ayurveda Ayatanam**. This repository contains the source code for the website showcasing Dr. Tejas Bharadwaj's Ayurvedic practice.

## Project Overview

This static website is designed to provide information about Ayurveda, the clinic's philosophy, and Dr. Tejas Bharadwaj. It features a modern, responsive design with aesthetic elements like glassmorphism and smooth animations.

## File Structure

- **`about-ayurveda-ayatanam.html`**: The main page detailing the Ayurveda Ayatanam center and its services.
- **`about-dr-tejas-bharadwaj.html`**: A dedicated page introducing Dr. Tejas Bharadwaj, his background, and expertise.
- **`styles.css`**: The core stylesheet handling the visual design, including responsive layouts and custom animations.
- **`scripts.js`**: JavaScript file capable of handling interactive behaviors on the site.
- **`images/`**: Directory containing image assets used across the website.

## Getting Started

To view the project locally:

1.  **Clone the repository** or download the source files.
2.  **Open the HTML files**: Double-click on `about-ayurveda-ayatanam.html` or `about-dr-tejas-bharadwaj.html` to open them in your web browser.

## Technologies Used

- **HTML5**: Semantic markup for content structure.
- **CSS3**: Advanced styling including Flexbox, Grid, and CSS variables.
- **JavaScript**: For frontend interactivity.
